Profile creation information

The profile editor was used to create the following profiles

  • VBC_LowAudioQuality.prx
  • VBC_HighAudioQuality.prx
  • VBC_Lossless.prx

You can use the profile button in the application to review the settings that are used for these profiles. These profiles must be in the standard directory, which on one system is the following path

D:\Program Files\Windows Media Components\Encoder\Profiles
