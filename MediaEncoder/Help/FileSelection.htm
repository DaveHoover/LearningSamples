File Selection Details

When you put the focus into this control, the File Open dialog box will open. Select the input wav file that you want to use for the encoding process.

The output files generated will use this file name minus the extension as the base file name. The three output files will be named as follows

  • High compression. Base file name.wmv
  • Low compression. Base file name_high.wmv
  • Lossless compression Base file name_ll.wmv
