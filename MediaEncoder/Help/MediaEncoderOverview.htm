Summary:

This program is a first attempt to automate some of the media encoding tasks that are necessary to take VBC audio messages and prepare them for archiving, high quality computer playback, and web playback

With one click, you can accomplish three separate encoding tasks. The input file wav file is used for all three encodings. The information for the description and title will be applied to all three output encodings as well.

The total time to generate all three encoded files is about 7-9 minutes.
